Project Description

This project presents my skills in building websites with interactive parts as quiz and game. It consist of one web page. In this project I introduce the jumping spiders family, the basic information about them and a few amazing photos. The user can take part in the quiz on spiders and answer the question choosing one from the provided list of options. They can also play the game chasing the fly in the spider's web.

Technical Features

  • BROWSERS: Google Chrome – the most popular browser by Google, Firefox – open- source browser supported by Mozilla Foundation
  • HTML: Mark-up language. Provides the structure of website to the browser
  • CSS Cascading Style Sheets: Change colours, fonts, animation and transitions on the page
  • PROGRAMMING LANGUAGES: Javascript. Used by all browsers
  • FRAMEWORKS: Bootstrap. A user interface framework for building websites with HTML/CSS/Javascipt
  • LIBRARIES: jQuery. Code snippets that enable a large amount of functionality without the need for handcoding
  • PROTOCOLS: http. It is how the website gets to the browser
  • GIT: Distributed Version Control System
  • GITHUB: Version Control Repository
  • TEXT EDITOR: Sublime Text 3
  • DEBUGGING TOOL: Chrome DevTools, Firefox Developer Tools
  • OTHER:
    • Google fonts
    • Google CDN (Content Delivery Network) for jQuery

CONTENT DETAILS:

  • HTML elements: script, link, meta, title, div, p, h1-2, ul, li, span, anchor, form, select (drop-down list)
  • CSS selectors: id, class, elements
  • CSS properties: colour, background, border, text, font, list, image, cursor (pointer, url for image), position (relative/absolute), display, margin, padding, width, height, border-radius
  • jQuery: shorthand method – .click() and no shorthand method .on() to attach event handler to the element, setInterval(), clearInterval()
  • Javascript: preventDefault() to stop propagation, if/else statement, event type (click, submit), Math.random(), popup box (alert), display method (console.log)
