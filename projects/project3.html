Project Description

It is a hand-coded website with HTML5 semantic elements presenting background video and a simple animation. The website is fully responsive and the breaking point are specified with media query.

Technical Features

  • BROWSERS: Google Chrome – the most popular browser by Google, Firefox – open- source browser supported by Mozilla Foundation
  • HTML: Mark-up language. Provides the structure of website to the browser
  • CSS Cascading Style Sheets: Change colours, fonts, animation and transitions on the page
  • PROGRAMMING LANGUAGES: Javascript. Used by all browsers
  • LIBRARIES: jQuery. Code snippets that enable a large amount of functionality without the need for handcoding
  • PROTOCOLS: http. It is how the website gets to the browser
  • GIT: Distributed Version Control System
  • GITHUB: Version Control Repository
  • TEXT EDITOR: Sublime Text 3
  • GRAPHICS EDITOR: GIMP
  • DEBUGGING TOOL: Chrome DevTools, Firefox Developer Tools
  • OTHER:
    • OpenShot Video Editor - free open-source video editor for Linux
    • resizeimage.net - free online image resizer and compressor
    • Google CDN (Content Delivery Network) for jQuery
    • Favicon & App Icon Generator

Content Details

  • HTML5 semantic elements: header, section, main, nav, article, footer
  • HTML5 multimedia elements: video
  • HTML elements: script, link, meta, title, div, p, h1, ul, li, anchor, strong, span
  • HTML attributes: href, role, lang, alt, name, content, type, source, src, autoplay, loop
  • CSS selectors: id, class, elements
  • CSS properties: box-sizing, colour, background, border, text, font, line-height, list, image, display, position, margin, padding, width, height, overflow, letter-spacing, opacity, transition, content, animation, z-index, white-space
  • CSS3 Media Queries
  • jQuery: hover() method
