Project Description

<<<<<<< Updated upstream

The purspose of this project is to help the user to get to the nearest pharmacy using only the partial postcode of their current or desired location. For this projects I've used AJAX to get JSON with data of pharmacies in the UK and API for data.gov.uk dataset. The user needs to provide the partial postcode, click the find button and the results will be populated on the screen. I've used latitude and longitude property to position the pharmacy on the map. Also I've added full google.maps funcionality to find the best way to get to the desired location.
