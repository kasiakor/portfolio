Project Description

This project presents my CSS and HTML knowledge in creating a website from the ground up using mobile first approach. I have marked up the content with semantic HTML and used progresive enhancement approach adding technology in leyers. I have implemented efficient CSS architecture to improve maintainability and scalability of the code. Typographic scale and color contrast checker helped to improve readability of the content and made it more pleasing for the eyes.

Technical Features

  • BROWSERS: Google Chrome – the most popular browser by Google, Firefox – open- source browser supported by Mozilla Foundation
  • HTML: Mark-up language. Provides the structure of website to the browser
  • CSS Cascading Style Sheets: Change colours, fonts, animation and transitions on the page
  • PROGRAMMING LANGUAGES: Javascript. Used by all browsers
  • PROTOCOLS: http. It is how the website gets to the browser
  • GIT: Distributed Version Control System
  • GITHUB: Version Control Repository
  • TEXT EDITOR: Brackets
  • GRAPHICS EDITOR: GIMP
  • DEBUGGING TOOL: Chrome DevTools, Firefox Developer Tools
  • OTHER:
    • CSS reset
    • Markup Validation Service
    • Color contrast checker
    • Typographic scale
    • Google fonts
    • Favicon& App Icon Generator
